# Music Playlist Sharing System — Project Proposal

---

## 1. Project Title

**Music Playlist Sharing System (MPSS)**

---

## 2. Problem Statement

In today's digital era, music listeners frequently create personal playlists across various platforms. However, sharing curated playlists with friends, colleagues, or communities in a structured, collaborative environment remains fragmented. Existing solutions are tightly coupled to specific streaming services and do not offer a lightweight, platform-independent way to organize, share, and discover playlists.

There is a need for a self-hosted, web-based system that allows users to create playlists, add songs, share playlists publicly or privately, and discover playlists shared by other users — all through a clean, intuitive interface with proper access control.

---

## 3. General Objective

To design and develop a web-based Music Playlist Sharing System that enables users to create, manage, share, and discover music playlists in a collaborative and secure environment.

---

## 4. Specific Objectives

1. To implement user registration and authentication with role-based access control.
2. To enable users to create, read, update, and delete (CRUD) playlists and songs.
3. To allow users to share playlists as public or private.
4. To provide an admin dashboard for managing users and monitoring system activity.
5. To implement a responsive, professional user interface using Bootstrap and JSP.
6. To apply the Model-View-Controller (MVC) architecture for clean separation of concerns.
7. To use PostgreSQL as the relational database with properly normalized schemas.

---

## 5. Scope

### In Scope:
- User registration, login, and logout
- Role-based access control (Admin, Regular User)
- Playlist CRUD operations (Create, Read, Update, Delete)
- Song management within playlists
- Public/Private playlist visibility
- Playlist sharing and browsing
- Admin dashboard with user management
- Server-side input validation and error handling
- Responsive UI with custom color branding

### Out of Scope:
- Music file upload or streaming
- Integration with third-party APIs (Spotify, Apple Music)
- Real-time chat or messaging
- Mobile native applications
